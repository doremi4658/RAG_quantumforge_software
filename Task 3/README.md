# Задание 3 — Создание векторного индекса

## Параметры индексации
- **Модель эмбеддингов**: `intfloat/multilingual-e5-small` (размерность 384)
- **Размер чанка**: 300 символов, перекрытие 50
- **Источник данных**: 31 текстовый файл из папки `Task 2/knowledge_base` (модифицированная вселенная Star Wars с русскоязычными именами)
- **Количество чанков**: 57 (после разбиения)
- **Время индексации**: 0.63 секунды
